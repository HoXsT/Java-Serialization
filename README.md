# Лабораторна робота №2: Java Serialization

## Опис
Цей репозиторій містить вихідний код для лабораторної роботи №2. 
Мета роботи — практичне засвоєння різних стратегій серіалізації та десеріалізації об'єктів у мові Java.

## Структура проекту
Проект містить реалізацію двох завдань, кожне з яких розділено на три пакети відповідно до обраної стратегії:

1. **`task1` (Основне завдання: Бібліотека)**
   - `version1`: Стандартна автоматична серіалізація (інтерфейс `Serializable`).
   - `version2`: Часткова серіалізація з ручним керуванням та використанням `transient` для несеріалізованих класів.
   - `version3`: Повне ручне керування збереженням через інтерфейс `Externalizable`.

2. **`extratask` (Додаткове завдання: Навчальний заклад)**
   - Аналогічні три версії, але з імплементацією простого алгоритму шифрування (XOR) для захисту чутливих даних (балів студента) у версії 2.

## Як запустити
Кожна версія має свій власний клас із методом `main()` для демонстрації роботи. Наприклад, щоб перевірити першу версію бібліотеки, запустіть клас `task1.version1.LibraryDriver`. Скомпільований код створить файли формату `.dat` у кореневій директорії.
